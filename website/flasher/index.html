Flash via Serial

Use esptool.py to flash the firmware to your device

pip install esptool

# For r8n8:
esptool.py --chip esp32s3 --port /dev/ttyUSB0 --baud 460800 \
    write_flash 0x0 flash_r8n8.bin

# For r8n16:
esptool.py --chip esp32s3 --port /dev/ttyUSB0 --baud 460800 \
    write_flash 0x0 flash_r8n16.bin

# For r16n16:
esptool.py --chip esp32s3 --port /dev/ttyUSB0 --baud 460800 \
    write_flash 0x0 flash_r16n16.bin
Web Flasher: Browser-based flashing requires HTTPS. For local development, use esptool.py above.
