Manual Flash (esptool.py)

If the web flasher doesn't work, use the command line:

pip install esptool

# Download firmware from releases, then:
esptool.py --chip esp32s3 --port /dev/ttyUSB0 --baud 460800 \
    write_flash 0x0 flash_r8n8.bin
Requirements: Chrome or Edge browser (Web Serial API). Hold the BOOT button on your ESP32-S3 when clicking Flash.
